# Claude Code Skills

A collection of Claude Code skills I use for Developer RElations

## Available Skills

| Skill | Description |
|-------|-------------|
| [devadvocate-blogger](devadvocate-blogger.md) | Write technical blog posts with an expert developer advocate voice. Produces content that balances deep technical expertise with approachable teaching, including proper syntax highlighting, links to official documentation, and hands-on examples. You need to change the .md to suit your company. |
| [cfp-hunter](cfp-hunter.md) | Fetch links for open CFPs which match the developer profiles your company is targetting. |

## Installation

To use these skills in your Claude Code setup, copy the skill files to your Claude Code skills directory.
